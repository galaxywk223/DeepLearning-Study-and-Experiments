- [cnn-mnist.md](cnn-mnist.md)：从卷积的数学直觉出发，理解 CNN 如何处理图像任务
- [transformer-self-attention.md](transformer-self-attention.md)：围绕 Q / K / V 与缩放点积注意力解释自注意力机制
