# 学习路线与项目导航

这份文档不是重复首页，而是告诉第一次进入仓库的人：应该怎么看，先看哪里，后看哪里。

## 这个仓库怎么读

整个仓库按三层组织：

1. 根目录 [README](../README.md)
   先快速看懂仓库主题、四条主线和代表结果。
2. 项目页
   看具体实验的定位、指标、图表和运行方式。
3. 笔记页
   看完整原理推导、实现细节和学习过程。

## 推荐路线

### 第一站：图像分类基础

- [MNIST 实验](../projects/01-mnist-cnn-experiments/README.md)
  适合第一次进入仓库的人。
  这里重点看 MLP 和 CNN 的差别，以及训练闭环是怎样搭起来的。

- [CIFAR-10 CNN 实验](../projects/02-cifar10-cnn-experiments/README.md)
  适合想看“同一任务上如何逐步提升性能”的读者。
  这里重点看 baseline、工程优化版和 ResNet 之间的差别。

### 第二站：进入 Transformer

- [字符级 Transformer 实验](../projects/03-char-transformer-experiments/README.md)
  适合已经理解分类任务，准备进入语言模型的人。
  这里重点看 bigram 到 Transformer 的升级，以及最小语言模型的实现骨架。

### 第三站：走向更真实的 GPT 工作流

- [子词级 GPT 实验](../projects/04-subword-gpt-experiments/README.md)
  适合已经理解最小 Transformer，希望继续理解 tokenizer、padding 和采样控制的人。
  这里重点看工程形态如何从教学版模型推进到更像真实 GPT。

## 如果你更偏向某一种阅读方式

## 结果资源说明

- 仓库只保留 `assets/showcase/` 下的精选展示资源。
- 原始数据、完整权重和训练输出目录默认作为本地运行目录处理，不纳入版本控制。
- 这样既保留了对外展示所需的证据，也避免公开目录被大量运行残留淹没。
