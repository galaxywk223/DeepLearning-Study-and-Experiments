# 学习路线与项目导航

这份文档用来说明这个仓库的阅读顺序。

## 怎么读

整个仓库按三层组织：

1. 根目录 [README](../README.md)
   先看整体内容和代表结果。
2. 项目页
   看具体实验结果、图表和运行方式。
3. 笔记页
   看原理和实现细节。

## 推荐路线

### 第一站：图像分类基础

- [MNIST 实验](../projects/01-mnist-cnn-experiments/README.md)
  先看 MLP 和 CNN 的差别，以及最基本的训练流程。

- [CIFAR-10 CNN 实验](../projects/02-cifar10-cnn-experiments/README.md)
  再看 baseline、改进版和 ResNet 之间的差别。

### 第二站：进入 Transformer

- [字符级 Transformer 实验](../projects/03-char-transformer-experiments/README.md)
  这里主要看 bigram 到 Transformer 的升级，以及最小语言模型的实现骨架。

### 第三站：走向更真实的 GPT 工作流

- [子词级 GPT 实验](../projects/04-subword-gpt-experiments/README.md)
  最后看 tokenizer、padding 和采样控制是怎么接进 GPT 训练流程里的。

## 如果你更偏向某一种阅读方式

- 想先看结果：直接看 [实验结果总览](./实验结果总览.md)
- 想先看教程：直接看 [笔记索引](../notes/README.md)
- 想先跑代码：直接进入任一 `projects/*/README.md`
